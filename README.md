# 🎧 MP3X

> Lightweight music downloading for the real internet.

---

## 🌍 Built For

MP3X is designed specially for:

- 📱 Keypad phone users
- 🌐 Opera Mini browsers
- 🛰️ Slow internet connections
- 💾 Low storage devices
- 🎵 Offline music listeners
- ⚡ Lightweight web experience lovers

---

## ✨ Philosophy

Modern music websites became:
- heavy
- ad-filled
- slow
- confusing

MP3X stays simple.

```txt
Open website
↓
Select music
↓
Download

Nothing else.
```
