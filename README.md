рҹ§  Smart Filename Parsing

Just name your file correctly вҖ” MP3X does the rest.

```python
  "Billie-Jean.mp3"                    вҶ’  "Billie Jean"
  "Village-Ghost-Story.mp3"            вҶ’  "Village Ghost Story"
  "I-Thought-I-Saw-Your-Face.mp3"      вҶ’  "I Thought I Saw Your Face"
```

**Coming soon вҖ” tag prefix support:**

```python
  "[NEW]-Ghost-Story.mp3"              вҶ’  рҹҶ•  Ghost Story
  "[SPECIAL]-Village-River.mp3"        вҶ’  вӯҗ  Village River
```

Rules applied automatically:
- Strip `.mp3` extension
- Replace `-` with spaces
- Title Case the result
- Preserve numbers
- Strip `[TAG]` prefixes into metadata
